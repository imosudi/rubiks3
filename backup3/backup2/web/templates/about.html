This application provides Rubik’s Cube details.

The total number of cublets making up an nXnXn Rubik’s cube. It also provides the total number of cublets making tu the faces of Rubiks’s cube. The faces are Up, Down, Front, Back, Right and Left. It provides the total number of hidden cublets of any given Rubik’s cube. It can also create smaller Rubik’s cube if possible by removing the faces. The application is, currently, allowed to process nXnXn Rubik's Cube (1 < n < 15). Made with Python3 and Flask framework backend, and HTML/CSS/JavaScript frontend .
